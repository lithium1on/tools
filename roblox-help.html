    • method 2

    • step 1:

      create api key manually
      go on the website

    • step 2:

      click "Create API Key"
      the big blue button

    • step 3:

      enter the general information
      the name which is required and optionally the description

    • step 4:

      enter the access permissions
      we require only 1 permission, select "assets" then select "write" in "Select Operation to Add"

    • step 5:

      skip the "Security", directly hit "Save & Generate Key"
      the big blue button

    • step 6:

      toggle on "I understand the security risks" and click "Save & Generate Key" again
      we dont log your api keys anyways

    • step 7:

      click "Copy Key To Clipboard" and save it somewhere!
      roblox doesn't let you see your key again after that

    • done!

      you got your api key ready
      ready to use
